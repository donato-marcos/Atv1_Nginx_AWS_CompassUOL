Bem-vindo à Atividade 1

Configuração de um servidor web com monitoramento automatizado.

Etapas do Projeto

  • Etapa 1: Configuração do Ambiente AWS (VPC, Sub-redes, EC2).
  • Etapa 2: Instalação e configuração do servidor Nginx.
  • Etapa 3: Criação de um script de monitoramento com notificações.
  • Etapa 4: Testes e documentação no GitHub.

Sobre o Projeto

Este projeto faz parte da Atividade 1 do programa de bolsas DevSecOps, com o objetivo de desenvolver habilidades em Linux, AWS e automação de processos.

O projeto consiste em configurar um servidor web na AWS, instalar o Nginx, criar uma página HTML simples e implementar um script de monitoramento que envia notificações em caso de indisponibilidade do serviço.
